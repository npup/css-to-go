Elements

The elements that receive some special treatment are:

Headings, Paragraphs, Hyperlinks, Lists, Form controls, and Buttons.

Headings

Headings of level 1 - 4 are supported.

Paragraphs

Paragraphs have some styles of their own to enchance readability. It is mostly a question of

  • Line height
  • Max-width

Hyperlinks

Links have some text-decoration and colors for hover/focus.

Behold - a link to an example page. And another one, to the same page.

Lists

Ordered and unordered lists are supported. In short, they are treated like this:

  • The list items keep any left-margin they may have
  • There is a hover style applied to the items
  • Ordered list items retain their list-style-type

Ordered list

  1. One potato
  2. Two potatoes
  3. Three potatoes
  4. Four

Unordered list

  • John
  • Paul
  • George
  • Ringo

Form controls

Some common form control elements are supported:

  • Text input
  • Text field
  • Radio button
  • Select
  • Checkbox
  • Submit- and "button" button

Buttons

Buttons of type button or submit and inputs of type submit are styled to stand out some, and also have some hover-focus indication to them.

CSS To Go

(This is the overview page. Also look at the examples of the standard theme and the light theme).

Sometimes, I write a bit of plain HTML and then start giving it some minimal styling. Recently, I’ve been thinking more and more like this:

If only there was a small CSS file that I could drop into the project just to get it over with instead!

— so I created CSS To Go.

What does it look like?

What it does

Basically, it is this:

  1. Some element styling
  2. A small number of utility classes

It is a quite small set of global CSS that are easy to apply to most any site.

How to use it

To enable any of these styles, wrap your content with the class .ctg-root (putting it all the way up on the <body> tag works just fine).

<!DOCTYPE html>
<html>
<head>
    <title>Example</title>
    <link rel="stylesheet" href="/path/to/css-to-go.css">
</head>
<body class="ctg-root">

    ... your content

</body>
</html>

Element styling

Most of its business is applying CSS to tag names, so no classes at all need be applied there. These elements get some help:

  • Headings (levels 1-4)
  • Paragraphs
  • Lists (ordered and unordered)
  • Hyperlinks (they look like this.)
  • Form controls with labels
  • \<em> and \<blockquote> elements

Most of these can be seen on this very page.

Utility classes

There are also a small number of utility classes:

  • .ctg-form-controls--row and .ctg-form-controls--col — for layout of form controls
  • .ctg-note — a “notification” type visual
  • .ctg-general-yes and .ctg-general-no — to signal “positive” and “negative” signals, respectively
  • .ctg-formatted-date — to format dates: 2021-06-15
  • .ctg-formatted-numerical — to format numbers in general: 125 56
  • .ctg-container — to be used as a top level container

(Granted, those two last formatters are merely applying some monospace to the font.)

Utility examples

Notes

The available types of “notes” are:

Plain note: .ctg-note

Please read this

Info note: .ctg-note--info

You have more data available

Warning note: .ctg-note--warn

Look out ahead!

Error note: .ctg-note--error

Stop doing that!
